## Installation

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNEME/nex-crm
```

2. Navigate into the project:

```bash
cd nex-crm
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```
